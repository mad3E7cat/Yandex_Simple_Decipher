# Yandex_Simple_Decipher
A simple tool with a small functional was written for the Yandex programming contest.
Functional:
1. Caesar cipher 
2. Morze code
3. Gronsfeld cipher
4. Digital ciphers
5. Atbash
6. Vertical permutation(INACTIVE)
7. Bacon cipher
8. Vigenere cipher(INACTIVE)
9. Frequency analysis(INACTIVE)
10. Keyboard code
11. ASCII-encoding
12. Solitaire
